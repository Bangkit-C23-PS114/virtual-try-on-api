# virtual-try-on-api
How to run our apps: 
1. Create .env file with format :
<br> DB_HOST= 
<br> DB_DATABASE= 
<br> DB_USER= 
<br> DB_PASSWORD= 
<br>*Our env are on documentation below
2. Run npm update
3. Run npm start
4. Use API Based on Documentation
